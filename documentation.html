Inits gloom. Required to be called. Preferrably called first before any other methods in main().

gloInit(width, height, title, fullscreen)

(int) width: the width in pixels of the window
(int) height: the height in pixels of the window
(const char *) title: the title of the window
(bool) fullscreen: whether the window is fullscreen or not

returns: (GLFWwindow*)

example:

GLFWwindow* window = glo::gloInit(1280, 720, "Glo Window", false);
Sets the clear color for OpenGL. By default (if not called) it is black, or (0.0f, 0.0f, 0.0f).

setClearColor(r, g, b);

(float) r: the amount of red in the clear color
(float) g: the amount of green in the clear color
(float) b: the amount of blue in the clear color

NOTE: VALUES RANGE FROM 0 TO 1; NOT FROM 0 TO 255.

returns: VOID

example:

glo::setClearColor(0.75f, 0.75f, 0.6f);
